Check updated dates of gov.uk content

  1. Create a .csv file with the paths you want checked (eg using Google Sheets)
  2. .csv must have a single column, with each row containing a path eg /foreign-travel-advice/spain, not a full URL like https://www.gov.uk/foreign-travel-advice/spain
  3. don't add a header row in the .csv, ie the first row should contain an actual path, not a title like 'Path'

After the file is uploaded, click Continue and a results .csv will be downloaded.

If any of the paths in the uploaded .csv were invalid, the corresponding row in the results will have the label 'error'.
